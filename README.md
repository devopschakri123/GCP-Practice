# RollDice Demo (Java 21 + Maven)

A simple Java-based RollDice demo application used for DevOps, CI/CD, and deployment demos.

## Build Instructions

### Build the project
```
mvn clean package
```

### Run the fat JAR
```
java -jar target/rolldice-0.1.0.jar
```

### Features
- JDK 21 compatible  
- Maven build  
- Runnable fat JAR using Shade Plugin  
- Simple RollDice console program
